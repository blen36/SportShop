\# SportShop



Учебное веб-приложение интернет-магазина спортивных товаров.



\## Описание проекта



SportShop — это клиент-серверное веб-приложение для покупки спортивных товаров через веб-интерфейс.  

Проект реализует каталог товаров, регистрацию и авторизацию пользователей, корзину, оформление заказов, оплату, личный кабинет и административную панель.



\## Технологии



\- Java 25

\- Jakarta Servlet API

\- JSP / JSTL

\- Maven

\- PostgreSQL

\- Apache Tomcat 11

\- HTML / CSS



\## Основные функции



\- Регистрация и авторизация пользователей

\- Каталог товаров

\- Поиск, фильтрация и сортировка товаров

\- Просмотр страницы товара

\- Корзина покупок

\- Оформление заказов

\- Имитация оплаты

\- Личный кабинет пользователя

\- Избранные товары

\- Отзывы и рейтинги

\- Сравнение товаров

\- Рекомендации товаров

\- Административная панель

\- Управление товарами

\- Управление заказами

\- Управление скидками и акциями

\- Складской учет

\- Отчеты по продажам

\## Настройка базы данных



Создать базу данных PostgreSQL:



```sql

CREATE DATABASE sportshop;

```



\## Запуск проекта



1\. Собрать проект через Maven.

2\. Скопировать `target/sportshop.war` в папку `webapps` Tomcat.

3\. Запустить Tomcat через `startup.bat`.

4\. Открыть приложение в браузере:



```text

http://localhost:8080/sportshop

```
