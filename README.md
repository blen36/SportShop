# SportShop

**SportShop** — учебное клиент-серверное веб-приложение интернет-магазина спортивных товаров.  
Проект разработан на Java с использованием JSP/Servlets, PostgreSQL и Apache Tomcat.

Приложение демонстрирует основные функции интернет-магазина: каталог товаров, регистрацию и авторизацию пользователей, корзину, оформление заказов, имитацию оплаты, личный кабинет и административную панель.

---

## Возможности проекта

### Для пользователя

- Регистрация и авторизация
- Просмотр каталога спортивных товаров
- Поиск, фильтрация и сортировка товаров
- Просмотр детальной страницы товара
- Добавление товаров в корзину
- Оформление заказа
- Имитация оплаты заказа
- Просмотр истории заказов
- Добавление товаров в избранное
- Отзывы и рейтинги
- Сравнение товаров
- Персональные рекомендации

### Для администратора

- Просмотр административной панели
- Добавление, редактирование и удаление товаров
- Управление категориями и складскими остатками
- Управление заказами и их статусами
- Работа со скидками и акциями
- Просмотр отчетов по продажам
- Отслеживание товаров с низким остатком

## Настройка базы данных

Создать базу данных PostgreSQL:

```sql
CREATE DATABASE sportshop;
```

После этого необходимо выполнить SQL-скрипт проекта для создания таблиц и добавления начальных данных.

---

## Настройка подключения к PostgreSQL

Подключение к базе данных выполняется через переменные окружения:

```text
DB_URL=jdbc:postgresql://localhost:5432/sportshop
DB_USER=postgres
DB_PASSWORD=your_password
```

При локальном запуске через Tomcat можно создать файл `setenv.bat` в папке `bin` Tomcat:

```bat
set "DB_URL=jdbc:postgresql://localhost:5432/sportshop"
set "DB_USER=postgres"
set "DB_PASSWORD=your_password"
```
---


## Запуск проекта

1. Собрать проект с помощью Maven.
2. Скопировать файл `target/sportshop.war` в папку `webapps` Apache Tomcat.
3. Запустить Tomcat через `startup.bat`.
4. Открыть приложение в браузере:

```text
http://localhost:8080/sportshop
```
